金融 及 汽车 板块 拖累 欧股 大跌 【 MarketWatch 伦敦 3 月 30 日讯 】 北京 时间 3 月 31 日 凌晨 消息 ， 周一 欧洲 股市 大幅 下滑 。 美国政府 拒绝 救援 通用汽车 和 克莱斯勒 两家 公司 以及 通用汽车 和 标致 汽车 CEO 均 被迫辞职 的 消息 ， 加重 了 市场 对 汽车 制造业 前景 的 担心 情绪 。 泛欧 道琼斯 600 指数 ( ST : SXXP ) 下跌 3.8% ， 收于 170.45 点 ， 完全 吐 出 上周 所获 涨幅 ， 而且 下跌 点数 超过 上周 上涨 点数 。 银行 板块 成为 拖累 泛欧 道琼斯 600 指数 下跌 的 最 主要 原因 。 德意志银行 ( Deutsche   Bank )   ( DE : DBK )   ( DB ) 跌幅 超过 11% ， 西班牙 国际 银行 ( Banco   Santander )   ( ES : SAN )   ( STD ) 跌幅 超过 7% 。 随着 经济衰退 和 房地产 市场 状况 持续 恶化 ， 上周末 西班牙 的 储蓄银行 机构 Caja   Castilla - La   Mancha 和 英国 的 邓福姆林 建房 互助 协会 ( Dunfermline   Building   Society ) 等 欧洲 金融机构 相继 得到 西班牙 和 英国政府 救援 。 受此 影响 ， 今天 欧洲 股市 中 的 银行业 板块 大幅 下跌 。 此外 ， 有 报道 称 本周 召开 的 20 国 集团 首脑 峰会 似乎 不大可能 出台 新 的 协同 刺激 经济 措施 。 欧洲 各 主要 国家 股市 中 ， 德国 DAX   30 指数 ( DX : 1876534 ) 下跌 5% ， 收于 3 , 989.23 点 ， 4000 点 整数 关口 再度 宣告 失守 。 法国 CAC - 40 指数 ( FR : 1804546 ) 下跌 4.3% ， 收于 2 , 719.34 点 。 英国 富时 100 指数 ( UK : UKX ) 下跌 3.5% ， 收于 3 , 762.91 点 。 周一 亚洲 股市 大幅 下滑 ， 东京 日经 225 平均 指数 重挫 4.5% ， 香港 恒生指数 下跌 了 4.7% 。 美国股市 开盘 大幅 走低 。 白宫 表示 可以 让 通用汽车 和 克莱斯勒 破产 并 强令 通用汽车公司 首席 执行官 瓦格纳 离职 的 消息 ， 令 投资者 情绪 低迷 。 受 美国 汽车 制造业 利空消息 的 影响 ， 意大利 菲亚特 ( Fiat )   ( IT : F ) 股价 下跌 了 8% 。 此前 菲亚特 公司 正在 考虑 与 克莱斯勒公司 结成 战略 联盟 。 法国 汽车 制造商 标致 汽车 公司 ( Peugeot )   ( FR : UG ) 下跌 了 9.2% 。 据 报道 ， 该 公司 首席 执行官 克里斯蒂安 - 斯特 莱夫 ( Christian   Streiff ) 于 上周日 被 废黜 ， 前 Corus 公司 首席 执行官 菲利普 - 瓦林 ( Philippe   Varin ) 将 在 6 月份 接替 其 职位 。 在 标致 汽车 公司 发生 高层 人事变动 之后 ， 美林公司 分析师 将 该 公司 的 股票 评级 调降 至 “ 中立 ” 。 其他 欧洲 汽车 公司 当中 ， 雷诺 汽车 公司 ( Renault )   ( FR : RNO ) 下跌 了 10.6% ， 宝马 汽车 公司 ( BMW )   ( DE : BMW ) 下跌 了 7.5% ， 戴姆勒 汽车 公司 ( Daimler )   ( DE : DAI )   ( DAI ) 下跌 了 9.2% 。
