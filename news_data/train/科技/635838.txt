阿尔卡特 朗讯 收购 互联网 API 目录 服务 新浪 科技 讯   北京 时间 6 月 29 日 晚间 消息 ， 据 国外 媒体报道 ， 阿尔卡特 朗讯 ( 以下 简称 “ 阿朗 ” ) 已经 收购 了 开放 互联网 应用 程序接口 ( API ) 目录 服务 ProgrammableWeb 。 ProgrammableWeb 收藏 了 超过 2000 个 开放 的 互联网 API ， 能 帮助 开发人员 开发 “ 混 搭 的 互联网服务 ” ( Mashup ) 。 随着 应用 商店 的 流行 ， 阿朗 等 公司 已经 意识 到 应用 和 开发者 的 重要性 ， 并 希望 成为 服务提供商 和 开发者 之间 的 中介 。 阿朗 全球 开发者 平台 副总裁 劳拉 · 莫林 ( Laura   Merling ) 表示 ， 阿朗 将 进一步 扩大 ProgrammableWeb 的 目录 ， 向 其中 加入 Open   API   Service 的 信息 。 Open   API   Service 是 阿朗 的 一款 电信 网络 API 目录 服务 ， 目的 在于 整合 不同 运营商 的 服务 ， 例如 基于 地理位置 的 服务 、 短信 和 广告 ， 使 开发者 无需 针对 不同 运营商 开发 不同 版本 的 应用 。 截至 周二 ， ProgrammableWeb 的 数据库 中有 2042 个 API ， 其中 最 流行 的 为 谷歌 地图 、 Flickr 和 YouTube 的 接口 。 在 该 网站 上 ， 用户 还 可以 查看 4900 个 整合 了 多个 不同 API 的 网站 或 应用 。 阿朗 没有 披露 这笔 交易 的 条款 ， 仅仅 表示 ProgrammableWeb 将 继续 独立 运营 。 ( 维金 )
