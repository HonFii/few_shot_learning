爱立信 实验室 网 中文版 上线   首个 开放 API 设备 商 飞象 网讯 ( 宗秀倩 / 文 ) 7 月 30 日 消息 ， 昨日 爱立信 宣布 ， 爱立信 实验室 网站 中文版 ( labs . ericsson . com / cn ) 正式 上线 。 作为 爱立信 新 推出 的 一个 Web 门户 ， 这个 门户 提供 了 一个 开放 的 环境 ， 可 与 应用程序 开发人员 、 技术 合作伙伴 和 消费者 共享 信息 。 爱立信 实验室 包含 了 一些 试用 引擎 和 应用程序 。 爱立信 也 成为 首家 开放 应用程序 API 的 设备 制造厂商 。 会上 ， 爱立信 中国 及 东北亚 区 首席 技术 官张 垦 表示 ， 爱立信 提出 ， 到 2020 年 全世界 有 500 亿 无线连接 ， 这些 连接 除了 人 ， 更 重要 的 是 物 与 物 之间 的 连接 ， 而 在 车辆 监控 、 公共设施 管理 、 公共安全 、 电力 及 能源 输送 等 垂直 行业 应用 方面 ， 成为 物 联网 连接 的 增长点 。 此番 爱立信 实验室 落地 中国 之 举 ， 中国 作为 最大 的 电信 市场 ， 最大 的 互联网 和 移动 互联网 市场 ， 拥有 最多 的 电信 基础设施 以及 快速 成长 的 物 联网 市场 ， 张垦 表示 ， “ 我们 希望 通过 爱立信 实验室 ， 了解 中国 用户 怎样 使用 互联网 ， 支持 运营商 的 发展 。 ” 爱立信 也 成为 首家 开放 应用程序 API 的 设备 制造厂商 。 与 网站 同时 上线 的 还 包括 十 多款 简单 好用 的 移动 应用 开发工具 ， 亦 称 API ( 应用 程序界面 ) 。 这些 工具 专 为 满足 中国 市场 的 需求 而 推出 ， 包括 备受 欢迎 的 脸部 识别 ( FaceDetector ) 、 移动 Java 通信 框架 ( MobileJavaCommunication   Framework ) 和 移动 定位 等 功能 。 再 结合 线上 应用 展厅 以及 诸如 论坛 、 博客 、 微博 和 邮件 列表 等 Web   2.0 功能 ， 开发者 将 如虎添翼 ， 为 中国 越来越 多 精通 技术 的 消费者 创建 更具 吸引力 的 应用程序 。 爱立信 认识 到 了 开发人员 所 陷入 的 困境 ， 并且 已经 采取 了 积极 的 措施 来 为 他们 提供 便利 。 例如 ， “ 开发者 社区 ” ( DeveloperConnection ) 项目 ( 原 “ 爱立信 移动 世界 ” 项目 ) 针对 运营商 网络 中 的 爱立信 产品 为 开发商 提供 了 软件开发 工具包 ( SDK ) 。 此外 ， 该 项目 还 能够 支持 第三方 应用 合作伙伴 与 众多 运营商 建立联系 。 据悉 ， 爱立信 实验室 简化 了 那些 实验性 开发人员 创建 应用 的 过程 。 它 还 帮助 开发者 与 运营商 接触 ， 简化 了 基本 电信 功能 的 实施 ， 并且 还 支持 他们 轻松 使用 高级 功能 。 爱立信 实验室 还 提供 了 服务器 基础设施 ， 使 开发者 可以 对 应用 进行 现场 测试 。   该 门户 的 两个 主要 部分 分别 是 开发人员 站点 ( 其中 包含 引擎 API 和 关联 的 资源 ) 以及 应用 展示区 ( 其中 包含 最终用户 应用 ) 。 一个 后 端系统 向 开发人员 提供 创建 、 发布 和 评估 应用 的 支持 和 工具 。 除了 服务 于 专业 的 先锋 开发者 社区 ， 爱立信 实验室 还 与 中国 顶尖 的 大学 开展 合作 ， 以 鼓励 创新 ， 同时 加快 这些 大学 科研成果 的 转化 。 譬如 ， 通过 爱立信 赞助 组建 的 致力于 ICT 行业 研究 与 学习 的 学生 社团 — — “ 爱立信 校园 俱乐部 ” ， 大学生 们 如今 可以 与 爱立信 实验室 的 专家 一同 工作 ， 充分利用 爱立信 实验室 网站 上 所 提供 的 各类 工具 和 功能 ， 联手 开发 移动 应用程序 。 爱立信 中国 及 东北亚 区 总裁 马志鸿 ( MatsHOlsson ) 表示 ： “ 我们 相信 ， 创新 绝 不仅仅 是 大 公司 、 大 企业 的 事 。 爱立信 实验室 网站 中文版 的 推出 ， 必将 向 世界 再次 证明 个人 和 中小企业 在 创新 领域 的 爆发力 。 我们 将 与 中国 的 开发者 一起 ， 向 运营商 提供 更 多 更具 吸引力 的 增值 应用 ， 进而 使 中国 乃至 全球 的 消费者 都 从中 受益 。 ” 今年 六月 ， 两名 中国 学生 凭借 其 开发 的 旅行 计划 应用程序 “ MobileTravelogue ( 移动 旅行家 ) ” ， 赢得 了 “ 爱立信 应用 大赛 ” 第三名 。 作为 一项 在 全球 范围 内 开展 的 学生 比赛 ， “ 爱立信 应用 大赛 ” 吸引 了 来自 28 个 国家 的 700 余名 注册 用户 和 120 支 注册 团队 参赛 。 参赛者 必须 利用 爱立信 实验室 网站 上 提供 的 API 来 开发 其 应用程序 。
