微软 推出 新 工具 可 将 iPhone 应用程序 转至 WP7 新浪 科技 讯   北京 时间 4 月 30 日 早间 消息 ， 微软 今天 通过 Windows   Phone 开发者 官方 博客 宣布 ， 公司 已经 发布 了 一套 新 的 工具 ， 该 工具 将 能够 帮助 开发者 将 iPhone 应用 程序转换 至 Windows   Phone   7 平台 上来 。 开发者 可以 登录 网站 以 进一步 了解 该 API 绘制 工具 ； 同时 微软 还 提供 了 一份 超过 90 页 的 白皮书 — — 《 致 iPhone 应用程序 开发者 的 Windows   Phone   7 平台 指南 ( Windows   Phone   7   Guide   for   iPhone   Applications   Developers ) 》 。 该 网站 还 配有 “ 开发者 的 故事 ” 等 视频 ， 以及 其他 帮助 开发者 着手 开发 时所 需要 的 各种 资料汇编 。 微软 特别 指出 ， 具体说来 这款 API 绘制 工具 的 功能 有点 类似 于 字典 ， 它 将 允许 开发者 选择 出 自己 的 iOS   API   call ， 然后 在 Windows   Phone   7 平台 上 搜索 相 类似 的 分类 、 方法 和 事项 通知 等 。 开发者 可以 搜索 指定 的 iOS   API   call ， 并 通过 C# 示例 代码 和 针对 两个 平台 的 API 文件 ， 在 Windows   Phone   7 平台 上 寻找 相同 的 类别 。 这些 示例 代码 能够 帮助 开发者 将 iOS 代码 转换 成为 C# 代码 。 在 一 开始 ， iOS   API 会 有 三个 分类 可以 通过 该 工具 进行 绘制 ： 网络 ( Network / Internet ) 、 用户界面 ( User   Interface ) 以及 数据管理 ( Data   Management ) 。 由于 平台 不同 ， 以及 用户界面 的 不同 ， 因此 并非 所有 的 API 均 可以 被 绘制 。 微软 保证 ， 即将 有 更 多 的 API 可以 通过 该 工具 进行 绘制 。 微软 也 表示 ， 针对 Android 应用程序 的 相关 转换 工具 也 在 加紧 开发 当中 。   ( 林静 )
