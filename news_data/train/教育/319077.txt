雅思 听力 场景 词汇 ： 熟知 词 更 要 熟悉 景 考 雅思 ， 要 掌握 重点 、 讲究 方法 。 比如 突破 雅思 听力 ， 朗阁 海外 考试 研究 中心 的 老师 给 大家 介绍 了 一些 方法 ： 不仅 要 熟知 听力 词汇 ， 还要 熟悉 场景 。 以 “ 住宿 ” 为例 ， 场景 中 不但 有 租房 、 学校 住宿 等 问题 ， 还 可能 包括 房屋 保险 、 搬家 、 房屋 设备 报修 等 延伸 内容 ， 如 细分 场景 中 的 词汇 ， 有 — — 相关 人士 ： houseagency ( 房产 代理 公司 ) ， houseagent ( 房产 代理人 ) ， landlord ( 房东 ) 等 ； 租房 类型 ： hallsofresidence ( 集体 宿舍 ) ， studenthostel ( 学生公寓 ) ， homestay ( 在 当地 居民 家中 住宿 ) 等 ； 房屋 地理位置 ： suburb = outskirts ( 郊区 ) ， countryside ( 郊外 ) ， urbanarea ( 市区 ) 等 ； 周边 设施 ： metro ( 地铁 ) ， parkinglots ( 停车场 ) ， fastfood ( 快餐厅 ) ， policestation ( 派出所 ) 等 ； 房屋 内部结构 ： kitchen ( 厨房 ) ， bathroom ( 浴室 ) ， storehouse ( 储藏室 ) 等 ； 房屋 设备 ： fridge ( 冰箱 ) ， exhaustfan ( 排气扇 ) ， shower ( 淋浴 ) ， wardrobe ( 衣橱 ) ， sofa ( 沙发 ) 等 ； 租金 和 押金 ： rentalprice ( 租金 ) ， deposit ( 押金 ) ， bills ( 账单 ) ， cheap ( 价格合理 ) 等 。 这个 场景 是 考题 的 重点 ， 要 全面 掌握 。
