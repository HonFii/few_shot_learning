19 日 北京自考 遇 马拉松赛 请 考生 提前 出行 北京 考试 报讯 ( 记者   马海丽 )   2008 年 北京 国际 马拉松赛 于 明天 上午 举行 ， 届时 ， 比赛 沿线 道路 将 交通 限行 。 市 自考办 提醒 赶考 的 考生 提前 出行 。 市 交管局 发布 ， 明天 上午 7 时 55 分至 比赛 结束 ， 14 条 路段 将 实行 交通 限行 。 具体 限行 时间 和 路段 是 ： 7 时 55 分 ， 由 前门 东站 经 天安门广场 东侧 路 ( 南向 北方 向 ) 至 天安门广场 东侧 路北口 ( 不 含 ) ， 社会 车辆 绕行 人民大会堂 西侧 路 和 正义路 。 7 时 58 分 ， 由 南池子 南口 经 西长安街 ( 东向西 方向 ) 至 复兴门 立交桥 ( 不 含 ) 。 8 时 10 分 ， 由 复兴门 立交桥 经西 二环路 东侧 辅路 ( 南向 北方 向 ) 、 月坛 北街 ( 双方 向 ) 、 三里河 路 ( 南向 北方 向 ) 至 甘家口 路口 ( 不 含 ) 。 8 时 18 分 ， 由 甘家口 路口 经 阜成路 ( 东向西 方向 ) 、 首体 南路 ( 南向 北方 向 ) 至 首体 南路 中口 ( 不 含 ) 。 8 时 26 分 ， 由 首体 南路 中口经 车公庄 西路 ( 东向西 方向 ) 、 玲珑 路 ( 东向西 方向 ) 至 慈寿寺 桥 ( 不 含 ) 。 8 时 35 分 ， 由 慈寿寺 桥经 蓝靛 厂 南路 ( 南向 北方 向 ) 至 游泳场 路口 ( 不 含 ) 。 8 时 50 分 ， 由 游泳场 路口 经 蓝靛 厂 北路 ( 北向南 方向 ) 、 蓝靛 厂路 ( 东向西 方向 ) 至 蓝靛 厂路 西口 ( 不 含 ) ， 昆明湖 东路 ( 南向 北方 向 ) 、 新建 宫门 路 ( 双方 向 ) 。 8 时 58 分 ， 由 蓝靛 厂路 西口 经 北四环路 南侧 辅路 ( 西向东 方向 ) 、 万泉河 路 西侧 辅路 ( 北向南 方向 ) 至 稻香 园 桥下 ( 不 含 ) 。 9 时 10 分 ， 由 稻香 园桥经 海淀 南路 ( 西向东 方向 ) 、 知春路 ( 西向东 方向 ) 至学知 桥下 ( 不 含 ) 。 9 时 23 分 ， 由学知 桥下 经 学院路 ( 南向 北方 向 ) 、 学清路 ( 南向 北方 向 ) 至林大 北路 东口 ( 不 含 ) 。 9 时 33 分 ， 由林大 北路 东口经 石板房 南路 ( 西向东 方向 ) 、 科荟路 ( 西向东 方向 ) 至林萃 路口 ( 不 含 ) 。 9 时 40 分 ， 由林萃 路口 经林 萃路 ( 南向 北方 向 ) 至林萃 桥下 ( 不 含 ) 。 9 时 44 分 ， 由林萃 桥下 经林萃 桥下 南侧 辅路 ( 西向东 方向 ) 、 奥林 西路 ( 北向南 方向 ) 、 北辰 西路 ( 北向南 方向 ) 至 民族 园 路口 ( 不 含 ) 。 10 时 05 分 ， 由 民族 园 路口 经 民族 园路 ( 西向东 方向 ) 至 奥林匹克 体育中心 西南 门 、 奥体 南路 ( 双方 向 ) 。 上述 时间 、 道路 除 公共 电 、 汽车 ( 小 公共汽车 除外 ) 和 持有 2008 年 恒源祥 北京 国际 马拉松 比赛 专用车 证 的 车辆 外 ， 禁止 其他 车辆通行 。 自考办 有关 负责人 表示 ， 自考 是 国家级 统一 考试 ， 按规定 ， 开考 15 分钟 后 ， 考生 不能 再 进入 考点 。 因此 ， 参加 当天 考试 的 考生 要 提前 了解 赛事 期间 的 交通管制 、 公交线路 调整 情况 。 建议 考生 最 好比 平常 提早 1 小时 左右 出门 ， 别开 私家车 ， 并 预先 做好 应急 措施 ， 如 碰上 交通管制 绕行 其他 路线 等 ， 避免 延误 考试 。 更 多 信息 请 访问 ： 新浪 自考 频道   自考 论坛   自考 博客 圈 特别 说明 ： 由于 各 方面 情况 的 不断 调整 与 变化 ， 新浪网 所 提供 的 所有 考试 信息 仅供参考 ， 敬请 考生 以 权威部门 公布 的 正式 信息 为准 。
