语数外 物化 齐 上阵 “ 中学 神兽 大全 ” 出炉 晨报 记者   张   毅   实习生   胡轶人 文言文 、 函数 、 酸碱性 ， 这些 让 中学生 们 很 是 头痛 的 知识 于 近日 被 有 才 网友 炮 制成 “ 中学 神兽 大全 ” ， 其 幽默 的 解释 也 让 不少 刚 从 期中考试 “ 战场 ” 上 下来 的 学生 们 感同 深受 。 在 这部 网友 自制 的 “ 神兽 大全 ” 中 ， 语文 、 数学 、 英语 、 物理 、 化学 等 学科 中 的 常见 名词 被 作者 用 别出心裁 地 谐音 法取 了 相应 的 “ 神兽 ” ( 动物 ) 名 ， 并 在 一旁 的 注释 中 生动 地 描述 了 该 “ 神兽 ” ( 相关 学科 名词 、 题目 ) 在 考试 中 对于 学生 的 杀伤力 。 面对 “ 大全 ” 中 对于 语文 神兽 “ 文言 蚊 ” ( 文言文 ) 的 描述 — — “ 极其 毒 ， 一旦 被 咬 ， 你 就 离 挂 蝌 ( 挂科 ) 不远 了 ” ， 刚刚 经历 高一 期中考试 的 王 同学 感叹 道 ， “ 该 神兽 威力 的确 了 得 ， 它 认识 我 ， 我 可不 认识 它 呀 ！ ” 那么 具体 都 有 哪些 神兽 呢 ？ 记者 为 你 列举 一二 — — 语文 神兽 ● 墨蝎 ( 默写 ) ： 极其 凶狠 ， 如果 你 不幸 被 墨蝎 ( 默写 ) 蜇 了 一下 的话 ， 那么 你 将 会 遇到 更加 凶狠 的 潮 蝎   ( 抄写 ) 。 ● 坐 蚊 ( 作文 ) ： 语文 界 中 最为 强悍 的 神兽 ， 腿法 了 得 ， 一旦 被 坐 蚊 ( 作文 ) 偏 踢 ， 那 … … ● 词语 解狮 ( 解释 ) ： 此 乃 凶物 ， 一旦 招惹 此物 ， 将会 遭到 文言 蚊   ( 文言文 ) 疯狂 的 报复 。 数学 神兽 ● 代鼠 ( 代数 ) ： 虽然 不如 函鼠 ( 函数 ) 凶猛 ， 但是 它 也 是 挂 蝌 ( 挂科 ) 的 远房 大 外甥 。 ● 相似 猩 ( 相似形 ) ： 有时 比较 温和 ， 但是 如果 它 发起 飙 ， 那么 你 将 受到 代鼠 ( 代数 ) 等 神兽 的 攻击 。 ● 桶 鸡 ( 统计 ) ： 看似 温和 ， 其实不然 。 他 是 中雪参 ( 中学生 ) 在 死亡 之 路 遇到 的 最后 两个 敌人 之一 ， 另 一个 则 是 桶 鸡 ( 统计 ) 同父异母 的 兄弟 ， 叫做 概驴 ( 概率 ) 。 英语 神兽 ● 英格烈 狮 ( English — — 英语 ) ● 帕特 兔   ( part2 — — 英语考试 第二 部分 ) ： 时而 温和 ， 腿法 高超 ， 发疯 后 你 将 收到 选择 踢   ( 选择题 ) 和 填空 踢 ( 填空题 ) 的 双重 攻击 。 物理 神兽 ● 欧姆 定驴 ( 欧姆定律 ) ： 极其 凶狠 ， 会 放电 ， 常常 和 电猪 ( 电阻 ) 一起 玩 。 ● 惯猩 ( 惯性 ) ： 冲击力 极大 ， 可以 直接 将 中雪参 ( 中学生 ) 冲到 挂 蝌 ( 挂科 ) 家 。 化学 神兽 ● 酸碱 猩 ( 酸碱性 ) ： 非常 可怕 ， 会 放出 酸 、 碱 两种 液体 ， 虽然 大部分 时间 被 中 和 掉 了 ， 可 依旧 威力 无穷 。
