江苏 2010 年 自考 调整 部分 专业 计划 通知 各市 自考办 ( 考试院 、 招考 中心 ) ， 各县 ( 区 ) 自考办 ， 有关 主考 学校 ： 根据 全国考委 有关 文件精神 ， 结合 我省 实际 ， 从 2010 年 5 月 起 ， 我省 高等教育 自学 考试 有关 专业 考试 计划 及 部分 专业课程 将 进行 调整 ， 请 按照 调整 后 的 课程 设置 与 学分 开展 毕业生 审核 等 相关 工作 。 现将 有关 问题 通知 如下 ： 一 、 采购 与 供应 管理 ( 专科 段 ) 专业 中 的 “ 05361 物流 数学 ” 课程 可用 “ 00020 高等数学 ( 一 ) ” 课程 替代 ： “ 05362 物流 英语 ” 课程 可用 “ 00012 英语 ( 一 ) ” 课程 替代 ： “ 05364 物流 企业会计 ” 课程 可用 “ 00041 基础 会计学 ” 课程 替代 。 采购 与 供应 管理 ( 本科段 ) 中 的 “ 05374 物流 企业 财务管理 ” 课程 可用 “ 00067 财务 管理学 ” 课程 替代 。 二 、 在 中药学 ( 本科段 ) 专业 中 ， 非 自学 考试 的 中药学 专业 专科 毕业生 报考 本 专业 时 ， 于 2010 年 5 月 前 加考 通过 “ 03035 有机化学 ( 四 ) ” 和 “ 03047 分析化学 ( 二 ) ” 两门 课程 中 任意 一门 的 ， 可 免考 “ 02974 中药学 ( 一 ) ” 、 “ 03050 药理学 ( 三 ) ” 和 “ 03040 中药 鉴定 学 ” 中 的 任意 一门 。 三 、 对 护理学 ( 本科段 ) 专业 计划 作 适当 调整 ： 1 . 结合 护理 专业 考生 的 工作 实际 ， 将 “ 03009 精神障碍 护理学 ” 课程 调整 为 选考 课 ， 与 “ 03010 妇产科 护理学 ( 二 ) ” 和 “ 03011 儿科 护理学 ( 二 ) ” 作为 一组 选考 课 ， 考生 需 从中 任意 选取 两门 报考 。 按原 计划 参加考试 的 考生 在 以下 5 门中 任选 3 门 即可 ： 762 心理卫生 、 03009 精神障碍 护理学 、 03007 急救 护理学 ( 或 767 急救 护理学 ) 、 03010 妇产科 护理学 ( 二 ) ( 或 765 妇产科 护理学 ) 、 03011 儿科 护理学 ( 二 ) ( 或 766 儿科 护理学 ) 。 2 . 将 自学 考试 专业 考试 计划 简编 ( 2008 版 ) 中 护理学 ( 本科段 ) 说明 第 4 点 调整 为 ： 自学 考试 护理 专业 专科 毕业生 不得 重复 报考 或 申请 免考 专科 阶段 已 通过 的 课程 ； 自考 护理 专业 专科 毕业生 按照 新 计划 考试 ， 若 在 专科 阶段 已 通过 “ 570 护理 管理学 ” 且 报考 本科段 时 按规定 换考 通过 “ 00488 健康 教育学 ” 课程 的 ， 可 免考 “ 03005 护理 教育 导论 ” 课程 。 若 其 在 专科 阶段 已 通过 “ 570 护理 管理学 ” 课程 ， 但 未换 考 过 “ 00488 健康 教育学 ” 课程 的 ， 在 报考 本 专业 时 ， 只能 选考 “ 03005 护理 教育 导论 ” 课程 。 四 、 经 全国 考办 批准 ， 对 动画设计 专业 ( 本科段 ) 部分 课程 做 以下 调整 ： 将 “ 03511 中外 电影史 ” ( 4 学分 ) 课程 调整 为 “ 11733 动画 视听 语言 ” ( 4 学分 ) 。 原 通过 中外 电影史 课程 的 考生 可以 免考 动画 视听 语言 课程 。 将 “ 03514 空间设计 ” ( 3 学分 ) 课程 调整 为 “ 01160 动画 运动 规律 ( 实践 ) ” ( 4 学分 ) 。 原 通过 空间设计 课程 的 考生 可以 免考 动画 运动 规律 ( 实践 ) 课程 。 将 “ 04845 互动 媒体 设计 ( 含 实践 ) ” ( 3 + 2 学分 ) 课程 调整 为 “ 11734 互动 媒体 设计 ( 一 ) ( 实践 ) ” ( 5 学分 ) 。 原 通过 互动 媒体 设计 课程 的 理论 和 实践 考核 中任 一部分 的 考生 均 可 免考 新 的 互动 媒体 设计 ( 一 ) ( 实践 ) 课程 。 请 各地 按 以上 通知精神 ， 认真 做好 宣传 、 组织 和 落实工作 。
