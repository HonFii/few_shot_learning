辽宁 2010 年 4 月 自考 将 开考 七个 新 专业 专科 段 开考 软件工程 、 会展 设计 与 管理 等 两个 专业 ， 软件工程 专业 的 主考 院校 为 大连 东软 信息 学院 ； 会展 设计 与 管理 专业 的 主考 院校 是 辽宁 师范大学 ； 独立 本科段 开考 汽车 维修 与 检测 、 船舶 与 海洋工程 ( 航海技术 方向 ) 、 船舶 与 海洋工程 ( 轮机 工程 方向 ) 、 采矿工程 、 珠宝 及 材料 工艺学 等 五个 专业 ， 汽车 维修 与 检测 专业 的 主考 院校 是 辽宁 工业 大学 ； 船舶 与 海洋工程 ( 航海技术 方向 ) 、 船舶 与 海洋工程 ( 轮机 工程 方向 ) 专业 的 主考 院校 是 大连海事大学 ； 采矿工程 、 珠宝 及 材料 工艺学 专业 的 主考 院校 是 辽宁 工程技术 大学 。 这 七个 专业 原则上 每年 4 月 和 10 月 进行 两次 考试 ， 如 省内 统一 加考 ， 则 随着 加考 的 时间 进行 考试 。 以上 专业 中 ， 软件工程 、 会展 设计 与 管理 、 采矿工程 、 珠宝 及 材料 工艺学 等 四个 专业 面向社会 开考 ， 汽车 维修 与 检测 、 船舶 与 海洋工程 ( 航海技术 方向 ) 、 船舶 与 海洋工程 ( 轮机 工程 方向 ) 等 三个 专业 不 接纳 零散 考生 报名 ， 由 各市 招考办 在 规定 的 时间 内 接受 省 招考办 批准 的 助学 单位 考生 集体 报名 。
