2011 年 北京自考 行业 委托 开考 专业 报考 有 限制 北京 考试 报讯 ( 实习 记者     徐   晗 )     10 月 自考 开考 专业 中 ， 部门 委托 开考 专业 由 委托 开考 单位 负责 集体 办理 考生 注册 、 报考 手续 ， 社会 零散 考生 不能 自行 网上 注册 报考 。 这是 记者 从 北京 教育 考试院 自考办 了解 到 的 。 据悉 ， 本次 开考 的 行业 委托 开考 专业 共有 27 个 ， 其中 部分 专业 考生 不能 单独 注册 、 单独 报考 。 这 类 专业 有 21 个 ， 分别 是 民航 服务 与 管理 ( 专科 ) 、 律师 ( 专 、 本科 ) 、 劳动 和 社会保障 ( 专 、 本科 ) 、 影视 表演艺术 ( 专科 ) 、 影视 美术设计 ( 专科 ) ， 地铁 通信 系统 ( 专科 ) 、 地铁 供用电 技术 ( 专科 ) 、 地铁 电动 客车 技术 ( 专科 ) 、 地铁 运输 与 管理 ( 专科 ) 、 调查 与 分析 ( 专 、 本科 ) 、 采购 与 供应 管理 ( 专 、 本科 ) 、 物流 管理 ( 专 、 本科 ) 、 项目管理 ( 专 、 本科 ) 、 人力 资源管理 ( 专 、 本科 ) 等 ， 除 停考 的 律师 专业 ( 专科 ) 不再 接受 新生 注册 外 ， 考生 注册 以上 专业 要 与 相关 行业协会 联系 。 餐饮 管理 ( 专 、 本科 ) 、 机电 一体化 工程 ( 专 、 本科 ) 、 护理学 ( 专 、 本科 ) 等 6 专业 考生 不能 自行 注册 ， 但 已 办理 注册手续 的 考生 可 自行 报考 。 机电 一体化 工程 专业 ( 专 、 本科 ) 注册 满 3 年 的 考生 要 由 机电 控股 工程师 培训中心 重新 认证 方 能 继续 报考 。 北京 教育 考试院 相关 负责人 提醒 ， 考生 想 报考 行业 委托 开考 专业 ， 在 报考 前 或 报考 期间 要 及时 与 相关 行业协会 取得联系 。
